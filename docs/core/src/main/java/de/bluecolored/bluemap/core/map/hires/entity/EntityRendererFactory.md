# EntityRendererFactory.java

## Přehled

`EntityRendererFactory` je funkcionální rozhraní (továrna) pro vytváření instancí `EntityRenderer`.

## Umístění

`de.bluecolored.bluemap.core.map.hires.entity.EntityRendererFactory`

## Metody

### `create(ResourcePack resourcePack, TextureGallery textureGallery, RenderSettings renderSettings)`

Vytvoří nový `EntityRenderer` pro vykreslování částí entit.
