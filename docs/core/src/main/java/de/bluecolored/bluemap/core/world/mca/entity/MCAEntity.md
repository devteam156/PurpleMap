# MCAEntity.java

## Přehled

`MCAEntity` reprezentuje entitu načtenou z MCA souboru (resp. z NBT).

## Umístění

`de.bluecolored.bluemap.core.world.mca.entity.MCAEntity`

## Implementovaná Rozhraní

`Entity`

## Atributy (z NBT)

- `id`: Typ entity.
- `uuid`: Unikátní identifikátor.
- `customName`: Vlastní jméno.
- `customNameVisible`: Viditelnost jména.
- `pos`: Pozice (Vector3d).
- `motion`: Pohyb.
- `rotation`: Rotace (yaw, pitch).
