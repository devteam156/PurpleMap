# VoidTileModel.java

## Přehled

`VoidTileModel` je implementace `TileModel`, která nedělá nic. Všechna data jsou zahozena.

## Umístění

`de.bluecolored.bluemap.core.map.hires.VoidTileModel`

## Použití

Používá se v případech, kdy chceme provést renderování (např. pro získání metadat pro low-res mapu), ale nepotřebujeme
ukládat výsledný 3D model.
