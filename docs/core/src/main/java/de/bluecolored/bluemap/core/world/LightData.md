# LightData.java

## Přehled

`LightData` je přepravka (data holder) pro informace o osvětlení bloku.

## Umístění

`de.bluecolored.bluemap.core.world.LightData`

## Atributy

- `skyLight`: Intenzita nebeského světla (0-15).
- `blockLight`: Intenzita umělého světla (0-15).

## Metody

Gettery a settery pro hodnoty světla.
