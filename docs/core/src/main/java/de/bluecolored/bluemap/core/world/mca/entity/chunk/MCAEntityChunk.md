# MCAEntityChunk.java

## Přehled

`MCAEntityChunk` je datová třída reprezentující chunk obsahující pouze entity (používáno v nových verzích MC, kde jsou
entity odděleny od bloků v souborech `entities/r.x.z.mca`).

## Umístění

`de.bluecolored.bluemap.core.world.mca.entity.chunk.MCAEntityChunk`

## Atributy

- `entities`: Pole entit v chunku.
- `dataVersion`: Verze dat.
- `position`: Pozice chunku.
