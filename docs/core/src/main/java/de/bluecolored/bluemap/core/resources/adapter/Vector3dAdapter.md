# Vector3dAdapter.java

## Přehled

`Vector3dAdapter` je GSON adaptér pro `Vector3d` (3D vektor s `double` souřadnicemi).

## Umístění

`de.bluecolored.bluemap.core.resources.adapter.Vector3dAdapter`

## Funkcionalita

- **Čtení:** Přečte JSON pole `[x, y, z]`.
- **Zápis:** Zapíše JSON pole `[x, y, z]`.
