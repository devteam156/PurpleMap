# AxisAdapter.java

## Přehled

`AxisAdapter` je GSON adaptér pro `Axis` enum (X, Y, Z).

## Umístění

`de.bluecolored.bluemap.core.resources.adapter.AxisAdapter`

## Dědičnost

`TypeAdapter<Axis>` -> `AxisAdapter`

## Funkcionalita

- **Čtení:** Přečte řetězec a převede ho na `Axis` (case-insensitive).
- **Zápis:** Zapíše název osy malými písmeny.
