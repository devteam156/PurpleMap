# Vector3fAdapter.java

## Přehled

`Vector3fAdapter` je GSON adaptér pro `Vector3f` (3D vektor s `float` souřadnicemi).

## Umístění

`de.bluecolored.bluemap.core.resources.adapter.Vector3fAdapter`

## Funkcionalita

- **Čtení:** Přečte JSON pole `[x, y, z]`.
- **Zápis:** Zapíše JSON pole `[x, y, z]`.
