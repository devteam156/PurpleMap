# EntityState.java

## Přehled

`EntityState` je obdoba `BlockState`, ale pro entity. Definuje, jak se má entita vykreslit. Tento formát není součástí
standardního Minecraft resource packu, ale je specifický pro BlueMap (v `assets/bluemap/entitystates`).

## Umístění

`de.bluecolored.bluemap.core.resources.pack.resourcepack.entitystate.EntityState`

## Atributy

- `parts`: Seznam částí (`Part`), ze kterých se entita skládá.
