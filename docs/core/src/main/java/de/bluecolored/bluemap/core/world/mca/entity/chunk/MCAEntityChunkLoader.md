# MCAEntityChunkLoader.java

## Přehled

`MCAEntityChunkLoader` je implementace `ChunkLoader` pro načítání chunků entit.

## Umístění

`de.bluecolored.bluemap.core.world.mca.entity.chunk.MCAEntityChunkLoader`

## Funkcionalita

- Dekomprimuje data.
- Deserializuje NBT data do objektu `MCAEntityChunk` pomocí BlueNBT.
