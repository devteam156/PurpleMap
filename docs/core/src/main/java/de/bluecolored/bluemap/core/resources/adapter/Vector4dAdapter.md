# Vector4dAdapter.java

## Přehled

`Vector4dAdapter` je GSON adaptér pro `Vector4d` (4D vektor s `double` souřadnicemi).

## Umístění

`de.bluecolored.bluemap.core.resources.adapter.Vector4dAdapter`

## Funkcionalita

- **Čtení:** Přečte JSON pole `[x, y, z, w]`.
- **Zápis:** Zapíše JSON pole `[x, y, z, w]`.
