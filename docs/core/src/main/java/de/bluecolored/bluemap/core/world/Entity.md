# Entity.java

## Přehled

`Entity` je rozhraní pro entitu ve světě (používané při renderování).

## Umístění

`de.bluecolored.bluemap.core.world.Entity`

## Metody

- `getId()`: Typ entity (např. `minecraft:pig`).
- `getUuid()`: Unikátní ID.
- `getPos()`: Pozice (Vector3d).
- `getRotation()`: Rotace.
