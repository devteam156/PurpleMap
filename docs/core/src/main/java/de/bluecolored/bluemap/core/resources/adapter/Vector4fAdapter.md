# Vector4fAdapter.java

## Přehled

`Vector4fAdapter` je GSON adaptér pro `Vector4f` (4D vektor s `float` souřadnicemi).

## Umístění

`de.bluecolored.bluemap.core.resources.adapter.Vector4fAdapter`

## Funkcionalita

- **Čtení:** Přečte JSON pole `[x, y, z, w]`.
- **Zápis:** Zapíše JSON pole `[x, y, z, w]`.
