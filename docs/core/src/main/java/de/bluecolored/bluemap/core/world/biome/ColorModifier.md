# ColorModifier.java

## Přehled

`ColorModifier` je funkcionální rozhraní pro úpravu barvy bloku na základě jeho pozice a vlastností.

## Umístění

`de.bluecolored.bluemap.core.world.biome.ColorModifier`

## Metody

### `apply(BlockAccess block, Color color)`

Upraví předanou barvu (`color`) na základě informací o bloku (`block`).
