# TextFormat.java

## Přehled

`TextFormat` je výčet (enum) stylů formátování textu, které Minecraft podporuje (kromě barev).

## Umístění

`de.bluecolored.bluemap.common.plugin.text.TextFormat`

## Hodnoty

- `OBFUSCATED` (k): "Magic" text (náhodně se měnící znaky).
- `BOLD` (l): Tučné písmo.
- `STRIKETHROUGH` (m): Přeškrtnuté písmo.
- `UNDERLINED` (n): Podtržené písmo.
- `ITALIC` (o): Kurzíva.

## Atributy

- `id`: Textový identifikátor formátu (např. "bold").
- `formattingCode`: Znak pro legacy formátování.

## Metody

Gettery pro atributy.
