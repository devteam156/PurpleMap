# LoadedAddon.java

## Přehled

`LoadedAddon` reprezentuje úspěšně načtený addon. Rozšiřuje třídu `Addon`.

## Umístění

`de.bluecolored.bluemap.common.addons.LoadedAddon`

## Dědičnost

`Addon` -> `LoadedAddon`

## Atributy

- `classLoader`: ClassLoader, kterým byl addon načten.
- `instance`: Instance hlavní třídy addonu.
