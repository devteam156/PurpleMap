# TextColor.java

## Přehled

`TextColor` je výčet (enum) základních barev používaných v Minecraft chatu a textových komponentách.

## Umístění

`de.bluecolored.bluemap.common.plugin.text.TextColor`

## Hodnoty

Obsahuje 16 standardních barev:
`BLACK`, `DARK_BLUE`, `DARK_GREEN`, `DARK_AQUA`, `DARK_RED`, `DARK_PURPLE`, `GOLD`, `GRAY`, `DARK_GRAY`, `BLUE`,
`GREEN`, `AQUA`, `RED`, `LIGHT_PURPLE`, `YELLOW`, `WHITE`.

## Atributy

- `id`: Textový identifikátor barvy (např. "red").
- `formattingCode`: Znak pro legacy formátování (např. 'c' pro červenou).

## Metody

Gettery pro atributy.
