# Addon.java

## Přehled

`Addon` je jednoduchá přepravka (data class) reprezentující nalezený (ale nutně ne načtený) addon.

## Umístění

`de.bluecolored.bluemap.common.addons.Addon`

## Atributy

- `addonInfo`: Informace o addonu (ID, verze, závislosti...).
- `jarFile`: Cesta k JAR souboru addonu.
