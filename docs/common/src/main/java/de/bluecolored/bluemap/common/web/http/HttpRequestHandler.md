# HttpRequestHandler.java

## Přehled

`HttpRequestHandler` je funkcionální rozhraní pro zpracování HTTP požadavků.

## Umístění

`de.bluecolored.bluemap.common.web.http.HttpRequestHandler`

## Metody

### `handle(HttpRequest request)`

Přijme `HttpRequest` a vrátí `HttpResponse`. Toto je hlavní bod integrace pro vlastní logiku webserveru.
