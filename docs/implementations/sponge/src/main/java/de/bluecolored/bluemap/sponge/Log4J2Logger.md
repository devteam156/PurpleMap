# Log4J2Logger.java

## Přehled

`Log4J2Logger` je implementace `AbstractLogger` pro Sponge. Přesměrovává logování do Log4j 2.

## Umístění

`de.bluecolored.bluemap.sponge.Log4J2Logger`

## Dědičnost

`AbstractLogger` -> `Log4J2Logger`

## Metody

Implementuje logovací metody voláním `org.apache.logging.log4j.Logger`.
